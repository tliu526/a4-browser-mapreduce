Welcome to Mapping With Friends!

Mapping With Friends is a collaborative in-browser implementation of the widely used MapReduce programming model of data processing. Click the 'Volunteer Resources' button to volunteer your computer's resources for a MapReduce job. Javascript functions will be embedded and run in your browser on portions of the job's data. After volunteering, you will be given an authentication token that allows you to submit your own MapReduce job! Simply click the 'Access Resources' button and provide a file with your map and reduce functions, along with the data you want to process.
